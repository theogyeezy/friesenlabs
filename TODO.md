## What is PARKED / NOT REAL YET
- **No web login flow** ‚Üí SPA runs in MOCK mode (`VITE_API_MOCK=1`); there is no `web/src/auth` directory, no Cognito Hosted UI domain, no OAuth flows configured on the app client.
- **AI/agent plane stubbed**: `agents/runtime.py` `ManagedAgentsRuntime` methods raise `NotImplementedError`; `/chat` 503 (`conversation_factory` returns `None`); executor is a noop.
- **Provisioning clients are stubs**: `api/prod_deps.py` wires `_StubStripe`, `_StubCognito`, `_Noop` (email/SMS/admin/agent_plane); `email_token_ok`/`sms_code_ok` hardcoded `False`; `_AccountStore` is in-memory.
- **Authored-but-unapplied modules**: cube, worker, observability, provisioning Lambda, cortex retrain target. Several modules (alb, api_service, api_cdn, web_hosting) are LIVE but **out-of-band / not in TF state**.
- **Drift**: SG rules (`api_from_alb`, ALB egress, ALB:80-from-CloudFront) + ALB SG + 2 IAM secret policies added via CLI, not in state; ECR `uplift-api` flipped to MUTABLE.
- **No domain/TLS-at-origin**: HTTPS is faked via CloudFront default cert over a plaintext-HTTP ALB; the ALB is internet-facing with only the shared CloudFront prefix-list as ingress (no shared-secret header).
